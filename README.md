# Gmail Cleaner (Python)

Tento skript som vytvorila ako svoj **prvý programátoský projekt**. Pomáha mi udržiavať poriadok v mailovej schránke automatickým mazaním správ od konkrétnych odosielateľov.

### Čo tento program dokáže:
* Pripojí sa k e-mailu cez zabezpečený protokol IMAP.
* Vyhľadá všetky e-maily od zadaného odosielateľa.
* Presunie ich hromadne do koša (Trash).

### Čo som sa naučila:
* Základy jazyka **Python** a prácu s knižnicou "imaplib"
* Ako bezpečne pracovať s **heslami aplikácií**  pre Google účet.
* Ako používať **GitHub** na zdielanie môjho kódu.

* Vytvorené s láskou k poriadku a čistému inboxu.*
